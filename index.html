Grand Circus Demo APIs

Donuts

  • https://grandcircusco.github.io/demo-apis/donuts.json
  • https://grandcircusco.github.io/demo-apis/donuts/INSERT_ID_HERE.json

The /donuts.json endpoint provides a list of donuts with limited information. The /donuts/SOME_ID.json endpoint provides more information about a specific donut, specified by inserting the donut's ID into the URL. The ref inside the /donuts.json endpoint is the URL for the specific endpoint for that donut.

Computer Science Hall of Fame

https://grandcircusco.github.io/demo-apis/computer-science-hall-of-fame.json

This JSON contains two different lists of some significant individuals in the history of computers. The first complete list is longer and has more detailed information. The second tiny list has fewer people and simpler information.
